Netty 4.x User Guide 中文翻译《Netty 4.x 用户指南》

Chinese translation of Netty 4.x User Guide . You can also see the demos of the guide here. There is a GitBook version of the book: http://waylau.gitbooks.io/netty-4-user-guide/ or http://www.waylau.com/netty-4-user-guide/ Let's READ!

《Netty 4.x 用户指南》中文翻译（包含了官方文档以及其他文章）。至今为止，Netty 的最新版本为 4.0.34.Final(2016-1-30)对此进行翻译，并在原文的基础上，插入配图，图文并茂方便用户理解。

作为提升，也推荐阅读《Netty 实战(精髓)》。与之类似的 NIO 框架还有 MINA，可以参阅《Apache MINA 2 用户指南》。
