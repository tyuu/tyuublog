Apache MINA User Guide 中文翻译《Apache MINA 2 用户指南》

Apache MINA User Guide 中文翻译《Apache MINA 2 用户指南》

Chinese translation of the Apache MINA User Guide (http://mina.apache.org/mina-project/userguide/user-guide-toc.html). The laset version of Apache MINA is 2.x. There is also a GitBook version of the book: http://waylau.gitbooks.io/apache-mina-2-user-guide. Let's READ!

《Apache MINA 2 用户指南》中文翻译。至今为止，Apache MINA 的最新版本为 2.0.12(2016-2-12)。利用业余时间对此进行翻译，并在原文的基础上，插入配图，图文并茂方便用户理解。如有勘误欢迎指正。

此外与 MINA 类似的 NIO 框架还有 Netty,可参阅《Netty 4.x 用户指南》 和 《Netty 实战(精髓)》。
